Paddle Catcher

In a way, the game is similar to pong, where the user has to control the paddle to keep the red ball from hitting the ground. However, for this game, the user is requiring to do more. With the paddle, the user also needs to catch the falling blue dot, preventing the blue surface from rising which can make it more difficult to keep the red ball off the ground. But, don’t worry, the height of the blue surface will reset after the user achieves a certain score. Lastly, beware as the red ball direction might change if it hit a blue dot.
